# Instagram Content Dashboard — 設計まとめ

**作成日**: 2026-05-15  
**ステータス**: 設計確定・実装前

---

## プロダクト概要

Instagram投稿の企画・制作を内製化するWebダッシュボード。
現在は代理店に依頼しているキャプション作成・画像構成を、AIを活用して自社で行えるようにする。

- **対象ユーザー**: 1人スタート → 後でチーム拡張
- **対象デバイス**: PC専用（モバイル対応は将来フェーズ）
- **モックアップ**: https://ig-dashboard-mockup.surge.sh

---

## 技術スタック

| 項目 | 選定 |
|------|------|
| フロントエンド | Next.js + TypeScript + Tailwind CSS + shadcn/ui |
| デプロイ | Vercel |
| DB | Supabase（Free枠） |
| 画像ストレージ | Vercel Blob |
| キャプション生成AI | Claude API（Anthropic）|
| 画像生成AI | DALL-E 3（OpenAI）|
| 自社投稿データ | Instagram Graph API |
| 競合投稿データ | Apify Instagram Profile Scraper |

## 各ペイン詳細

### A — 自社過去投稿
- Instagram Graph APIで自社アカウントの投稿を取得
- 3つの表示モード切替
  - **カード**: サムネイル＋キャプション冒頭1〜2行
  - **一覧**: キャプションのみのリスト
  - **カレンダー**: 月カレンダー上にサムネイル表示・年月選択・前後月ナビ

### B — 競合投稿
- Apify Instagram Profile Scraperで取得
- 5〜10社登録・アカウント切替
- 「更新」ボタン押し時のみデータ取得（オンデマンド）
- ホバーで「保存」ボタン → [競合 | 保存済み] タブで管理
- **投稿クリックでCペインの参照投稿として選択**

### C — AIキャプションエディタ
- **インプット①**: Bペインの競合投稿を参照として選択（任意）
- **インプット②**: 今回のテーマを一言入力（任意）
- Claude APIで3案同時提案（設定のブランドレギュレーションも渡す）
- 案を選んで編集、気に入らない点を入力して再提案
- 全ラウンド遡れる
- リアルタイムカウンター: 残り文字数（2,200字制限）＋ハッシュタグ数（30個制限）
- **「このキャプションで確定する」** → Dペインがアクティブ化
- **「やり直す」** → 確定解除

### D — 画像構成プランナー
- Cペインで確定するまでロック表示
- 単一画像 / カルーセルを選択
- DALL-E 3でたたき台画像を生成
- 生成画像はVercel Blobに自動保存（1時間失効対策）

---

## 設定モーダル（ヘッダーの「設定」から開く）

- **ブランドレギュレーション・トンマナ**: プロンプト形式の自由記述
  - CとDのAI出力に反映
- **外部ドキュメント読み込み**（将来対応）: Google Doc / Spreadsheet のURL

## ワークフロー

```
1. A/Bペインで自社過去投稿・競合投稿を参照
      ↓
2. B競合投稿を参照として選択（任意）＋テーマ入力（任意）
      ↓
3. Claude APIで3案提案 → 選ぶ → 編集
      ↓
4. 「このキャプションで確定する」
      ↓
5. DペインでDALL-E 3でたたき台画像生成
      ↓
6. 下書き保存
      ↓
7. （将来）Meta Business Suiteで投稿予約
```
